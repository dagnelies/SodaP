Let's take as example, the data from [mtgjson.com], which contains two data sets:

  • the list of all MTG cards
  • the list of all MTG sets

Since all our examples are based on it, becoming familiar with its structure will come in handy.

Let's start with the cards. They are stored in a dictionary called cards, looking like this:

{
...
"Nevinyrral's Disk" : { /* card data */ },
           "Norrit" : { /* card data */ },
"Nullmage Shepherd" : { /* card data */ },
...
}

And individual card data looks like this:

The sets are stored in a dictionary like this:

{
  "LEA" : { /* set data */ },
  "LEB" : { /* set data */ },
  "2ED" : { /* set data */ },
  ...
}

Each an individual set data looks like this:

{
  "name" : "Nemesis",
  "code" : "NMS",
  "gathererCode" : "NE",         // optional
  "oldCode" : "NEM",             // optional
  "magicCardsInfoCode" : "ne",   // optional
  "releaseDate" : "2000-02-14",
  "border" : "black",
  "type" : "expansion",
  "block" : "Masques",
  "onlineOnly" : false,
  "booster" : ["land", "common", "common", "common", /*...*/ , "uncommon", [ "rare", "mythic rare" ] ],
  "cards" : [ {}, {}, {}, ... ]
}
